# 🌍 SSAFY-WORLD
## 🤔프로젝트 한 줄 설명
SSAFY 인들을 위한 커뮤니티 서비스를 제공해주는 애플리케이션
## 👉프로젝트 목적
SSAFY에서 소통할 수 있도록 사용하는 애플리케이션에서는 같은 반과 캠퍼스에서만 커뮤니케이션을 활발히 할 수 있다.
